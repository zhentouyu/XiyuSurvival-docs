# 每日任务
| 命令 Command | 描述 Description |
|:-|:-|
| `/dq` | 打开每日任务 |

每天**早上6点**会刷新9个任务，分为简单、中等、困难3组，每组随机抽取3个。

完成当天所有任务，可获得500Points。

[关于点数](./points)

**被连锁挖下来的方块不计入每日任务。**
