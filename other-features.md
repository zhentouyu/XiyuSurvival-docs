# 其他

## 投票
| 命令 Command | 描述 Description |
|:-|:-|
| `/poll` | 查看所有投票 |

可能不定期会有投票，届时会在进服消息里打出来。

过一阵子会变成在网页投票，然后回服里拿奖励。
## 举报
| 命令 Command | 描述 Description |
|:-|:-|
| `/report <player> <reason>` | 举报玩家 |

理论上reason可以有候补选项，但是不知道为什么GUI不好用。

不过有举报建议还是和服主联系比较好，因为服主平时可能看不太见这个举报。
