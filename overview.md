# 服务器概述
欢迎来到 **Xiyu Survival**。这是一个以**长期生存**为基石，刚刚起步的 Minecraft 纯净生存社区。

我们欢迎任意类型的玩家，哪怕是刚接触Minecraft的萌新。

使用Java 版 1.21.11，Paper 服务端。

使用登录插件，无正版验证，降低游玩门槛。


## 服务器规定

为了其他玩家的游玩体验和服务器整体性能，我们制定了以下几条规则：

::: warning 不要做
* 破坏他人建筑
* 偷盗他人物品（部分可用[箱子锁](./blocklocker)避免被偷盗，部分容器可能需要漏斗输入但不希望输出，此时也请加上箱子锁，若被偷了请[找服主](./contact)解决）
* 未经他人同意进行PVP（请双方在游戏内消息来发起PVP请求，以便判断）
* 违规高速移动和飞行
* 破坏原版玩法（作弊，如防击退、防摔落、杀戮光环等）
:::

偷盗他人物品的惩罚是暂时允许受害者PVP偷盗者，扣除相应点数×1.5，返还相应物品。

::: tip 可以做
* 进行任意原版玩法和合理利用原版特性，但会造成计算严重卡顿的除外（如卡服机）
* 矿透，但尽可能标记一下这片区域(如火把)，或者挖走大多数矿物
* 投影
* 灵魂出窍(Free Camera)
:::
