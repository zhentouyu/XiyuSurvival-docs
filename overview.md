# 服务器概述
欢迎来到 **Xiyu Survival**。这是一个以**长期生存**为基石，刚刚起步的 Minecraft 纯净生存社区。

我们欢迎任意类型的玩家，哪怕是刚接触Minecraft的萌新。

使用Java 版 1.21.11，Paper 服务端。

使用登录插件，无正版验证，降低游玩门槛。


## 服务器规定

为了其他玩家的游玩体验和服务器整体性能，我们制定了以下几条规则，部分详见[服务器规则解释](./rules)：

::: warning 不要做
* 破坏他人建筑
* 偷盗他人物品（可用[箱子锁](./blocklocker)避免被偷盗，若还是被偷了请[找服主](./contact)解决），参见[关于偷盗的处理方案](./rules)
* 未经他人同意进行PVP（请双方在游戏内消息来发起PVP请求，以便判断）
* 违规高速移动和飞行
* 破坏原版玩法（作弊，如防击退、防摔落、杀戮光环等）
* 易造成服务器卡顿或崩溃的机器或玩法，如更新抑制器
:::

::: tip 可以做
* 进行任意原版玩法和合理利用原版特性，但会造成计算严重卡顿的除外（如卡服机）
* 矿透，但在260418后不发送不可见方块了，所以可能用不了矿透了
* 投影
* 灵魂出窍(Free Camera)
:::
