# 邮件
| 命令 Command | 描述 Description |
|:-|:-|
| `/mail inbox` | 打开收件箱 |
| `/sendmail <player>` | 发送邮件 |

发送邮件必须输入标题和内容，可选附件。

::: warning
需要注意的是，放入附件后，在发出邮件前不会扣除背包内物品，此时请勿把刚才的物品再次向附件投入，否则可能报物品不够，会扣除所有该物品且不返还，**找不回来**。我们尚不知道这是特性还是bug。
:::

你也可以按`G`来打开Mailbox菜单，其实此处的`G`是按键绑定的杂项-快捷操作键。

发送对象可以不在线，但必须在20260424后上过至少一次线。
