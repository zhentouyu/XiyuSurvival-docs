# Mineceaft NeoForge服
我们使用Minecraft Java 1.21.1，NeoForge21.1.233。

NeoForge服使用外置登录(Authlib-injector)，和Survival服不同。

NeoForge服地址：**forge.943689.top**

NeoForge QQ群：**980817391**

## 登录方式
请前往[LittleSkin](https://littleskin.cn)或[Fish-BSS皮肤站](https://skin.943689.top)注册一个账号。

::: tip
LittleSkin邮箱需要验证，Fish-BSS邮箱不验证，可乱填。

Fish-BSS可以注册在LittleSkin上被占用的角色。

在部分启动器上，可以很容易地通过点击一个按钮登录LittleSkin账号。
:::

对于HMCL和BakaXL，若使用LittleSkin登录，请见来自LittleSkin文档的[在客户端配置 Yggdrasil 外置登录](https://manual.littlesk.in/yggdrasil/client)。

或者，对于HMCL、BakaXL和PCL，把登录皮肤站后用户中心出现的有`将此按钮拖动至启动器`（Fish-BSS）或`将该卡片拖动至启动器以快速配置外置登录`（LittleSkin）字样的按钮拖动至启动器。

输入账号和密码后，点击登录，选择角色。

然后就可以进服了。

## 注意
Forge没有像survival一样成熟的区域回档和玩家回档方案。故请务必注意数据安全。

有用的paper插件将逐渐寻找替代mod。

服务器规则同[survival](./overview)，此外：

::: warning Forge服特有规则
* 所有机械动力机器必须套机壳
:::
