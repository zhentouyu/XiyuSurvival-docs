# 箱子锁
说是箱子锁，其实是容器锁和门锁。

在箱子上**直接**右键放置告示牌，就会自动上锁。

在标有`[私人]`的告示牌的第三行和第四行，您可以添加最多两位好友的名字，他们也可以访问这个箱子。想要添加更多人？在箱子旁边放置第二个告示牌，会自动在第一行添加 [更多]，在第二行添加 [红石]。如果您想更改这些设置，请右键单击告示牌进行编辑。

对于门，请在门上方方格的第一行放置一个写有`[私人]`的牌子。将牌子放在门下方方格甚至门上也可以，但看起来不太美观。

标签 [Everyone] 允许所有人使用。标签 [Redstone] 允许漏斗访问容器，并允许红石信号打开门。

还有一个 [Timer:X] 标签，其中 X 是 0 到 9 之间的数字。门（包括活板门）会在指定的秒数后自动关闭。使用 0 则门永远不会自动关闭。

事实上，你可以用中文或英文来添加箱子锁：
``` 
everyone:
    - 所有人
    - Everyone
more_users:
    - '[更多]'
    - '[More Users]'
private:
    - '[私有]'
    - '[Private]'
redstone:
    - [红石]
    - [Redstone]
golem:
    - [Golem]
    - [Golems]
timer:
    - [定时]
    - [Timer]
```
